Major Business Goal

Decrease transaction cost by a factor of a 100

Minting from 7 dollars to 7 cents

Why it is expensive to mint a domain

  • Blockchain is sequential in nature
  • Logarithmic growth of performance
  • Transactions perform a lot of operations

Ethereum performance

When new block is mined, all competitive miners are dropping their calculations, receive new block and start from scratch.

What are L2 solutions?

  • L2 is a blockchain on top of L1 blockchain (Ethereum in our case)
  • Tokens can be transfered to and from L2 like from or to a contract
  • Tokens are transferable within L2
  • "L" is for "Level"

Zero Knowledge Proofs

  1. ZK-Proof - a piece of cryptography that confirms certain operations have happened
    1. A Signature - is a proof that someone has a private key and approves a message
    2. A Proof can be generated from individual signatures for transactions making it a collective signature
    3. A Proof can contain information that certain permission checks were made
  2. Proofs can be much much longer than a signature
  3. Proof size grows slower than number of operations it confirms or even be fixed length
  4. One can verify the proof by only having a merkle root and program hash

Zero Knowledge Rollups

a solution that optimize for:

  1. Gas Usage
  2. Security

A Rollup

a batch of L2 transaction

stored on L1 in a form of state change + ZK-Proof

Minimum gas usage

  1. Batch user transactions when submitted to L1
  2. Only state change is published instead of full state
  3. Universal "proof" for multiple transactions
  4. Single contract custom blockchain as L2

Security

  1. Zero Knowledge Proofs
  2. L2 inherits security guarantees of L1

Downsides of rollups

Anything outside of gas cost and security becomes more complex

  • Read smart contract state
  • Generate signature in wallet
  • Listen for changes on L1
  • Contract on L2 is not upgradable
  • Token management with L1 and L2
  • Longer mining time (>20 minutes)

Logical Solution

  • L1: Registry that supports minting from L2
  • L1: a list of smart contracts to ensure security of L2
  • L2: a single smart-contract blockchain
  • L2: Registry Mirror 2.0
